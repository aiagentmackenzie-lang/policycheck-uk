# PolicyCheck UK

AI-powered insurance policy analysis for UK brokers and underwriters. Scan, analyse, and defend with compliance checking.

## Features

- **Policy Scanning**: Upload and analyze insurance policy documents
- **Claim Validation**: Check claim eligibility against policy terms
- **AI Analysis**: Automated compliance checking against UK regulations
- **Human Review**: Workflow for analyst approval and override
- **Regulatory Citations**: References Insurance Act 2015, FCA ICOBS

- SwiftUI
